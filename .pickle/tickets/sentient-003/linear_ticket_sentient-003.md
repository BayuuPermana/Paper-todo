# Description

## Problem to solve
Mouse-based navigation is inefficient for deep work.

## Solution
Implement a global keyboard listener:
- `j / k`: Move sidebar selection down/up.
- `n`: Focus the 'New Task' input.
- `Enter`: Open selected task.
- `Escape`: Clear all focuses.
