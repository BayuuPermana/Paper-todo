# Responsive Split Layout Implementation Plan

## Overview
Transform the single-column todo list into a 2/3 - 1/3 dual-pane dashboard. This involves expanding the main container, implementing a grid system, and ensuring the layout degrades gracefully on mobile.

## Current State Analysis
- `src/App.jsx`: Renders everything inside a single `.paper-container`.
- `src/index.css`: `.paper-container` is capped at 600px.

## Implementation Approach
1.  **Broaden the Canvas**: Increase the `max-width` of the main container to 1200px.
2.  **Grid Implementation**: Use CSS Grid on the main container to define the two segments.
3.  **Component Wrapper**: Wrap existing todo logic in a `.left-segment` div. Create a `.right-segment` div for the productivity tools.
4.  **Responsive Breakpoints**: Add media queries to stack segments on smaller screens.

## Phase 1: CSS Grid & Container Refactor
### Overview
Modify the CSS to support a wider, split layout.

### Changes Required:
#### 1. `src/index.css`
**Changes**: Update `.paper-container` and add segment styles.
```css
.paper-container {
  width: 95%;
  max-width: 1200px; /* Expanded for dual-pane */
  background-color: #fff;
  box-shadow: 5px 5px 15px rgba(0, 0, 0, 0.5);
  padding: 30px;
  border: 3px solid black;
  border-radius: 10px;
  position: relative;
  margin: auto;
  display: grid;
  grid-template-columns: 2fr 1fr;
  gap: 30px;
  align-items: start;
}

@media (max-width: 1024px) {
  .paper-container {
    grid-template-columns: 1fr;
    max-width: 600px;
  }
}

.left-segment {
  /* No specific styles needed yet, inherits from grid */
}

.right-segment {
  display: flex;
  flex-direction: column;
  gap: 20px;
  border-left: 2px dashed #ccc; /* Visual separation for the "desk" */
  padding-left: 30px;
}

@media (max-width: 1024px) {
  .right-segment {
    border-left: none;
    border-top: 2px dashed #ccc;
    padding-left: 0;
    padding-top: 30px;
  }
}
```

## Phase 2: App Structure Refactor
### Overview
Wrap existing components in the new segment divs.

### Changes Required:
#### 1. `src/App.jsx`
**Changes**: Introduce `.left-segment` and `.right-segment`.
```jsx
  return (
    <div className="paper-container">
      <div className="left-segment">
        <AddTodo onAdd={addTodo} />
        <DragDropContext onDragEnd={handleDragEnd}>
          <TodoList
            todos={todos}
            onToggleComplete={toggleComplete}
            onDelete={deleteTodo}
            onAddSubTask={addSubTask}
            onToggleSubTask={toggleSubTask}
            onDeleteSubTask={deleteSubTask}
          />
        </DragDropContext>
      </div>
      <div className="right-segment">
        {/* Placeholders for upcoming features */}
        <div className="calendar-placeholder">Calendar (Coming Soon)</div>
        <div className="timer-placeholder">Timer (Coming Soon)</div>
      </div>
    </div>
  );
```
