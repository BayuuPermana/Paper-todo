# Stationary Desk Protocol Implementation Plan

## Overview
Refactor the dual-pane layout to fix the container height, isolate scrolling to the task list, and ensure symmetrical vertical margins.

## Current State Analysis
- `src/index.css`: `.paper-container` grows with content. `body` handles scrolling.

## Implementation Approach
1. **Body Constrainment**: Prevent the body from scrolling.
2. **Container Sizing**: Fix `.paper-container` height to `calc(100vh - 100px)`.
3. **Left Segment Scroll**: Enable `overflow-y: auto` on `.left-segment`.
4. **Mobile Fallback**: Ensure the layout reverts to natural scrolling on small screens.

## Phase 1: CSS Refactor
### Overview
Apply the new height and scroll rules.

### Changes Required:
#### 1. `src/index.css`
**Changes**: Update `body`, `.paper-container`, and segments.
```css
body {
  overflow: hidden; /* Prevent page scroll */
  height: 100vh;
}

.paper-container {
  height: calc(100vh - 100px); /* Fixed height with 50px margins */
  overflow: hidden; /* Contain segments */
}

.left-segment {
  height: 100%;
  overflow-y: auto;
  padding-right: 15px; /* Space for scrollbar */
}

.right-segment {
  height: 100%;
}

@media (max-width: 1024px) {
  body {
    overflow: auto;
    height: auto;
  }
  .paper-container {
    height: auto;
    overflow: visible;
  }
  .left-segment {
    height: auto;
    overflow: visible;
  }
}
```
