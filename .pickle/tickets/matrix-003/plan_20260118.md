# Integrated Input Matrix Implementation Plan

## Overview
Refactor all task and subtask input areas into a unified 'Matrix' bar where buttons are integrated into the border of the input field.

## Current State Analysis
- `AddTodo.jsx`: Vertical layout with separate buttons.
- `TodoItem.jsx`: Simple input with adjacent buttons.
- `index.css`: Missing unified bar styles.

## Implementation Approach
1. **CSS Infrastructure**: Add `.input-matrix` and related styles to `index.css`.
2. **AddTodo Refactor**: Update the task sidebar input to use the matrix pattern.
3. **TodoItem Refactor**: Update the subtask input to use the matrix pattern.
4. **Symbol Update**: Replace 'Create' and '->' with '»'.
