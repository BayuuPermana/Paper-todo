# God Mode Paper Calendar Implementation Plan

## Overview
Create a custom, paper-styled calendar component to show the current month and today's date.

## Current State Analysis
- `App.jsx`: Has a placeholder for the calendar.
- `index.css`: Defines the global paper styles.

## Implementation Approach
1.  **Component Skeleton**: Create `src/PaperCalendar.jsx`.
2.  **Date Logic**: Use native `Date` object to calculate days in the month and the starting day of the week.
3.  **Grid Layout**: Render days in a 7-column grid.
4.  **Paper Styling**: Apply hand-drawn borders and handwritten fonts. Highlight the current day with a "scribble" or a solid circle.
