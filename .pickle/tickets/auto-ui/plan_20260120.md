# Autosave Indicator Implementation Plan

## Overview
Implement a visual "Saving..." indicator to provide user feedback during persistence operations.

## Current State Analysis
- `src/App.jsx` handles all state modifications and calls `db.js`.
- `src/db.js` operations are async.
- No visual feedback exists.

## Implementation Approach
1.  Create `SavingIndicator` component.
2.  Implement a `withSaving` HOC/wrapper in `App.jsx` to manage the `isSaving` state centrally and ensure minimum visibility duration.
