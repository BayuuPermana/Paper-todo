# Text Truncation Implementation Plan

## Overview
Implement CSS-based text truncation with ellipsis for task and subtask names, and slightly reduce the sidebar width for a tighter layout.

## Current State Analysis
- `index.css:11`: `.app-layout` has sidebar width at 350px.
- `index.css:191`: `.todo-text` has no truncation rules.
- `TaskSidebar.jsx:65`: Has some inline truncation on a wrapper div, but the inner text div might still wrap.
- `SubTaskItem.jsx`: Uses `.todo-text` but has no truncation enforcement.

## Implementation Approach
1. Update `index.css`:
    - Set `.app-layout` sidebar width to 300px.
    - Update `.todo-text` to include `overflow: hidden`, `text-overflow: ellipsis`, and `white-space: nowrap`.
2. Update `TaskSidebar.jsx`:
    - Apply truncation to the specific task name `div`.
3. Update `TodoItem.jsx`:
    - Ensure the main task button truncates.
